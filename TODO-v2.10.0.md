# NAS-OS v2.10.0 开发计划

**创建日期**: 2026-03-14
**目标发布日期**: 2026-03-14

## 🎯 版本目标

v2.10.0 聚焦于 **文件共享服务核心功能**，实现 SMB/CIFS 和 NFS 共享支持。

**职责**: 软件工程、系统架构

- [ ] SMB 服务管理器 (`internal/smb/manager.go`)

- [ ] SMB 共享 CRUD 操作
- [ ] 用户权限映射
