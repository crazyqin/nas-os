## 二、模块功能概述

### 1. 配额管理模块 (`internal/quota/`)

配额管理模块提供完整的存储配额生命周期管理，包括：

| 功能 | 文件 | 描述 |
|------|------|------|
| 配额类型 | `types.go` | 用户配额、用户组配额、目录配额三种类型 |
| 配额管理 | `manager.go` | 配额创建、更新、删除、查询 |
| 使用监控 | `monitor.go` | 实时监控配额使用，多级预警告警 |
| 趋势分析 | `trend.go` | 使用量趋势追踪、增长预测、模式检测 |
| 清理策略 | `cleanup.go` | 按年龄、大小、模式、配额比例、访问时间清理 |
| 报告生成 | `report.go` | JSON/CSV/HTML 多格式报告导出 |
| 智能优化 | `optimizer/optimizer.go` | 自动调整建议、违规检测、优化报告 |

**核心特性**：
- 软限制/硬限制双阈值机制
- 4级预警体系（info 60%/warning 70%/critical 85%/emergency 95%）
- 告警升级机制（未处理自动升级通知）
- 宽限期管理
- 配额调整历史记录

### 2. 监控模块 (`internal/monitor/`)

系统资源监控模块提供全方位的基础设施监控：

| 功能 | 文件 | 描述 |
|------|------|------|
| 系统监控 | `manager.go` | CPU、内存、Swap、运行时间、负载 |
| 磁盘监控 | `disk_health.go` | SMART 信息、健康状态、温度 |
| 指标收集 | `metrics_collector.go` | 周期性指标采集、趋势数据 |
| 健康评分 | `health_score.go` | 综合健康评分系统 |
| 告警引擎 | `alert_rule_engine.go` | 可配置告警规则 |
| Prometheus | `prometheus.go` | Prometheus 指标导出 |

**指标收集能力**：
- 采集间隔：每分钟
- 数据保留：约 1 周（10080 个数据点）
- 扩展指标：备份统计、磁盘健康状态

### 3. 成本报告模块 (`internal/reports/`)

专门的成本分析模块，提供全面的成本管理能力：

| 功能 | 文件 | 描述 |
|------|------|------|
| 成本计算 | `storage_cost.go` | 完整成本模型计算 |
| 成本优化 | `cost_optimizer.go` | 优化建议生成 |
| 成本分析 | `cost_analysis.go` | 增强成本分析 |
| 电力成本 | `electricity_cost.go` | 电力消耗成本追踪 |
| 财务报告 | `financial_report.go` | 财务报表生成 |
| 资源报告 | `hubu_resource_report.go` | 综合资源报告集成 |

**成本模型构成**：
```
总成本 = 容量成本 + IOPS成本 + 带宽成本 + 电力成本 + 运维成本 + 折旧成本
```
