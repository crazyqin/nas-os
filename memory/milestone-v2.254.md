# v2.254 里程碑规划

**规划日期**: 2026-03-24
**目标版本**: v2.254.290

## 🎯 核心目标

基于竞品分析（群晖DSM 7.3、飞牛fnOS 1.0），本版本聚焦于**网盘原生挂载**和**AI能力增强**。

---

## 📋 功能规划

### P0 - 必须完成

| 功能 | 模块 | 说明 | 参考 |
|------|------|------|------|
| 网盘原生挂载 | `internal/cloudsync` | 支持115、夸克、阿里云盘主流网盘直连 | 飞牛fnOS |
| AI数据脱敏 | `internal/ai` | 敏感信息本地处理后发送AI | 群晖AI Console |
| 远程访问优化 | `internal/remote` | 改善远程访问稳定性和速度 | 飞牛FN Connect |

### P1 - 重要功能

| 功能 | 模块 | 说明 | 参考 |
|------|------|------|------|
| 文件请求 | `internal/share` | 允许外部用户上传文件到指定目录 | 群晖Drive |
| 共享标签 | `internal/files` | 文件标签系统，支持团队协作 | 群晖Drive |
| 安全风险指标 | `internal/security` | KEV/EPSS/LEV风险评分集成 | 群晖DSM 7.3 |

### P2 - 增强功能

| 功能 | 模块 | 说明 | 参考 |
|------|------|------|------|
| 应用商店完善 | `internal/appstore` | 开发者SDK、应用审核流程 | 飞牛fnOS |
| 文件锁定 | `internal/files` | 协作时防止编辑冲突 | 群晖Drive |
| 邮件系统调研 | - | 评估是否需要MailPlus类似功能 | 群晖 |

---

## 🔧 技术债务

1. **测试覆盖率**: 目标从当前 ~60% 提升至 75%
2. **文档更新**: API文档同步更新
3. **性能优化**: 大文件上传断点续传优化

- 新增功能测试覆盖率 > 80%
