## 🚀 新增功能

### 应用商店增强 (App Store)
参考飞牛fnOS应用市场设计，新增以下功能：

- **模板版本管理** - 支持多版本模板存储、版本查询、弃用和删除
- **应用更新检测** - 自动检查 Docker Hub 镜像更新，识别预发布版本
- **应用备份恢复** - 配置导出/导入、数据卷备份、校验和验证
- **健康检查** - 容器状态、TCP端口、HTTP端点检测，资源使用监控

### 智能相册管理 (Photos)
参考群晖 Photos 功能设计：

- **照片自动分类** - 按人物/地点/时间/场景自动分类
- **智能相册** - 系统相册（收藏、最近、视频、自拍、截图）+ 条件匹配系统
- **照片去重** - 基于感知哈希的相似图片检测
- **缩略图缓存** - LRU策略、多尺寸支持、自动清理
- **EXIF增强** - 完整EXIF信息提取（GPS/时间/相机）

### 文件锁定增强 (File Lock)
- 新增冲突解决模块 (`conflict.go`)
- 版本管理增强 (`version.go`)
- 完善测试覆盖
