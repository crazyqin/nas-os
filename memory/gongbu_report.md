## 🐳 Dockerfile 分析

### Dockerfile (生产镜像)

**优点**:
- 多阶段构建，镜像大小约 20-25MB
- UPX 压缩进一步减小 30-50%
- BuildKit 缓存挂载加速构建
- OCI 标签完整
- 健康检查配置

**配置详情**:
```
基础镜像: alpine:3.21
Go 版本: 1.26-alpine
压缩工具: UPX --best --lzma
端口: 8080, 445, 139, 2049, 111, 20048
健康检查: 30s 间隔，10s 超时，3 次重试
```

### Dockerfile.dev (开发镜像)

**优点**:
- Air 热重载支持
- Delve 调试器集成
- 完整开发工具链

**注意**:
- 镜像较大（包含所有开发依赖）
- 适用于开发环境，非生产部署
