### 2.2 高危问题详情

#### G115 整数溢出 (73 处)
**位置**: quota/optimizer, optimizer, backup, monitor, disk 等模块
**风险**: 可能导致内存分配错误或数值计算错误
**示例**:
- `internal/quota/optimizer/optimizer.go:548` - uint64 -> int64 转换
- `internal/disk/smart_monitor.go:726` - uint64 -> int 转换

#### G304 路径遍历 (216 处)
**风险**: 用户输入未验证直接用于文件操作，可能导致任意文件读写
**建议**: 使用 `filepath.Clean()` 验证路径，限制在安全目录内

#### G204 命令执行 (175 处)
**风险**: 使用 `exec.Command` 执行外部命令，可能导致命令注入
**建议**: 避免用户输入直接拼接到命令中，使用参数化调用

**Go 标准库/官方包漏洞**:

- net/http 拒绝服务漏洞

### 4.2 建议

1. **升级 quic-go** - 多个高危 DoS 和 Panic 漏洞
2. **升级 gin** - 日志注入和请求走私风险
3. **升级 gorilla/websocket** - 整数溢出
4. **检查 x/image 使用** - TIFF 解析可能导致 DoS
