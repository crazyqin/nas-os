## 📖 升级指南

### 从 v2.59.0 升级

#### Docker 部署

```bash
# 停止旧容器
docker stop nasd
docker rm nasd

# 拉取新镜像
docker pull ghcr.io/crazyqin/nas-os:v2.60.0

# 启动新容器
docker run -d \
  --name nasd \
  --restart unless-stopped \
  -p 8080:8080 \
  -v /data:/data \
  -v /etc/nas-os:/config \
  ghcr.io/crazyqin/nas-os:v2.60.0
```

#### 二进制部署

```bash
# 下载新版本
wget https://github.com/crazyqin/nas-os/releases/download/v2.60.0/nasd-linux-$(uname -m)

# 替换旧版本
sudo systemctl stop nasd
sudo mv nasd-linux-$(uname -m) /usr/local/bin/nasd
sudo chmod +x /usr/local/bin/nasd
sudo systemctl start nasd
```
