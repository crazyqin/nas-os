## 📦 下载

### 二进制文件

```bash
# AMD64 (x86_64)
wget https://github.com/crazyqin/nas-os/releases/download/v2.77.0/nasd-linux-amd64

# ARM64 (Orange Pi 5, Raspberry Pi 4/5)
```
