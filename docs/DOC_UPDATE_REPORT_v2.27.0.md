#### docs/api.yaml (OpenAPI 规范)
- 版本号更新：2.20.1 → 2.27.0
- 新增媒体服务 API：
  - GET /media/stream/{id} - 获取媒体流
  - POST /media/transcode - 创建转码任务
  - GET /media/transcode/{taskId} - 获取转码状态
  - GET /media/thumbnail/{id} - 获取缩略图
  - GET /media/subtitles/{id} - 获取字幕列表
  - POST /media/subtitles/{id}/upload - 上传字幕
- 新增配额自动扩展 API：
  - GET /quota/auto-expand - 获取策略列表
  - POST /quota/auto-expand - 创建策略
  - GET /quota/auto-expand/{id} - 获取策略详情
  - PUT /quota/auto-expand/{id} - 更新策略
  - DELETE /quota/auto-expand/{id} - 删除策略
  - GET /quota/auto-expand/{id}/history - 扩展历史
- 新增 Schema 定义：
  - AutoExpandPolicy - 自动扩展策略
