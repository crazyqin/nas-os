- 文件名使用小写，单词间用 `-` 连接
- 所有文档使用 Markdown 格式
- 日期格式：YYYY-MM-DD

### 模板
- 技术文档：参考 [architecture.md](./architecture.md) 模板
- API 文档：使用 OpenAPI 3.0 规范
- 会议记录：包含时间、参与人、议题、决议

### 更新流程
