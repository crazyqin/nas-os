# 应用中心 WebUI 优化方案

**设计部门**: 礼部（品牌营销部门）
**日期**: 2026-03-29

---

## 一、现状分析

**现有功能：**
- ✅ 基础搜索（名称/描述/标签匹配）
- ✅ 单选分类筛选
- ✅ 一键安装（无配置应用）
- ✅ 已安装应用管理
- ✅ 安装配置弹窗

**待优化问题：**
1. 搜索体验单一，缺少引导和反馈
2. 分类筛选不支持多选和组合
3. 安装过程无实时进度展示
4. 一键安装缺少智能推荐

---

## 二、分类筛选设计优化

### 2.1 多级分类导航
```
┌─────────────────────────────────────────┐
│ 📦 应用商店                              │
├─────────────────────────────────────────┤
│ 🔍 搜索框                                │
│                                          │
│ 分类筛选:                                │
│ ┌────┐ ┌────┐ ┌────┐ ┌────┐ ┌────┐     │
│ │全部│ │媒体│ │下载│ │开发│ │安全│ ...  │
│ │ 25 │ │ 8  │ │ 4  │ │ 5  │ │ 3  │     │
│ └────┘ └────┘ └────┘ └────┘ └────┘     │
│                                          │
│ 状态筛选: [已安装] [可一键安装] [需配置] │
│                                          │
│ 排序: [热门] [最新] [评分] [名称]        │
└─────────────────────────────────────────┘
```

### 2.2 实现要点
- **多选分类**：支持同时选择多个分类
- **状态快速筛选**：已安装、可一键安装、需要配置
- **排序切换**：热门度、最近更新、用户评分、名称
- **筛选持久化**：localStorage保存用户偏好

---

## 三、搜索功能优化

### 3.1 搜索体验增强
- 热门搜索标签：展示Top 5高频搜索词
- 搜索历史：localStorage保存最近10条
- 实时建议：输入时下拉匹配结果
- 快捷操作：建议项直接显示"安装"按钮

---

## 四、一键安装流程改进

### 4.1 智能配置推荐
- 端口冲突检测：自动推荐可用端口
- 存储路径智能检测：自动发现媒体目录
- 配置模板：一键填充推荐配置
- 配置验证：安装前检查配置有效性
