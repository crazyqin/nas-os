## 📦 升级指南

### 从 v2.58.0 升级

#### 方式一：二进制文件升级

```bash
# 停止服务
sudo systemctl stop nas-os

# 备份配置
sudo cp -r /etc/nas-os /etc/nas-os.bak

# 下载新版本 (根据架构选择)
wget https://github.com/crazyqin/nas-os/releases/download/v2.59.0/nasd-linux-amd64
chmod +x nasd-linux-amd64
sudo mv nasd-linux-amd64 /usr/local/bin/nasd

# 启动服务
sudo systemctl start nas-os

# 验证版本
nasd --version
```

#### 方式二：Docker 升级

```bash
# 拉取新镜像
docker pull ghcr.io/crazyqin/nas-os:v2.59.0

# 停止旧容器
docker stop nasd
docker rm nasd

# 启动新容器
docker run -d \
  --name nasd \
  --restart unless-stopped \
  -p 8080:8080 \
  -v /data:/data \
  -v /etc/nas-os:/config \
  ghcr.io/crazyqin/nas-os:v2.59.0

# 验证
docker logs nasd | head -5
```
