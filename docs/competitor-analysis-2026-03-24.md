# 竞品分析报告 - 2026-03-24

## 一、飞牛fnOS（国产黑马）

### 核心优势
1. **免费开源** - 基于Debian深度开发，完全免费
2. **现代化UI** - 界面精美，类似群晖DSM布局，上手容易
3. **Docker支持** - 可视化界面管理容器，支持Docker Compose
4. **影视库** - 自动刮削影片信息，支持蓝光杜比播放
5. **相册备份** - 原画质备份，多设备共享
6. **内网穿透** - 内置免费内网穿透服务
7. **RAID支持** - 支持RAID 0/1/5/Linear等
8. **安全防护** - 内置防火墙，权限分级

### 我们可借鉴
- [ ] 内置免费内网穿透服务
- [ ] 影视库自动刮削功能增强
- [ ] 可视化Docker管理面板优化

## 二、群晖DSM 7.3

### 核心优势
1. **Synology Tiering** - 冷热数据分层机制，类似我们已实现的Fusion Pool
2. **AI Console** - 去敏感机制，保护用户私密数据
3. **Synology Office** - 多人协同编辑
4. **Synology Drive** - 共享标签、文件请求、文件锁定功能
5. **Container Manager** - Node.js容器部署优化
6. **exFAT原生支持** - 简化相机SD卡、移动硬盘数据导入
7. **数据加密强化** - 重置系统时自动上锁加密存储

### 我们可借鉴
- [x] Tiering分层机制 - 已实现Fusion Pool
- [ ] AI去敏感机制
- [ ] 文件锁定功能（协同编辑冲突规避）
- [ ] exFAT原生支持

## 三、TrueNAS Scale 24.10 "Electric Eel"

### 核心优势
1. **Docker迁移** - 从Kubernetes切换到Docker，简化应用部署
2. **RAIDZ Expansion** - 单盘扩展RAIDZ vdev
3. **Docker Compose** - 完整支持自定义YAML配置
4. **TrueCloud备份** - 数据备份和恢复到云存储
5. **全局搜索** - WebUI全局搜索功能
6. **ZFS快速去重** - 实验性功能，未来版本默认启用
7. **NVMe S.M.A.R.T** - NVMe驱动器测试支持
8. **SMB审计** - 客户端和文件事件审计
