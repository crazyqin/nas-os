### 🛡️ 兵部 - FTP/SFTP 服务器模块

**任务**: 实现传统文件传输协议支持

**功能清单**:
- [ ] FTP 服务器基础实现
  - [ ] 被动/主动模式支持
  - [ ] 匿名登录支持
  - [ ] 用户认证集成
  - [ ] 虚拟目录映射
- [ ] SFTP 服务器基础实现
  - [ ] SSH 密钥认证
  - [ ] 用户权限隔离
  - [ ] chroot 目录限制
- [ ] WebUI 配置界面
  - [ ] FTP/SFTP 开关
  - [ ] 端口配置
  - [ ] 带宽限制
  - [ ] 日志查看

**API 端点**:
```
GET    /api/v1/ftp/config           # FTP 配置
PUT    /api/v1/ftp/config           # 更新 FTP 配置
GET    /api/v1/ftp/status           # FTP 服务状态
POST   /api/v1/ftp/restart          # 重启 FTP 服务

GET    /api/v1/sftp/config          # SFTP 配置
PUT    /api/v1/sftp/config          # 更新 SFTP 配置
GET    /api/v1/sftp/status          # SFTP 服务状态
POST   /api/v1/sftp/restart         # 重启 SFTP 服务
```

**目录**: `internal/ftp/`, `internal/sftp/`  
**预计工时**: 3-4 天
