# 计费系统配置指南

**版本**: v2.61.0  
**更新日期**: 2026-03-15

## 概述

NAS-OS 计费系统提供资源使用统计、成本核算、账单管理功能，适用于多用户环境和资源运营场景。

## 功能特性

- **多资源计费**：存储、带宽、服务
- **灵活的定价模型**：阶梯定价、按存储池定价
- **自动账单生成**：支持日结、周结、月结、年结
- **用量统计与分析**：详细的用量报表
- **发票管理**：PDF 导出、公司信息配置

---

## 快速开始

### 1. 启用计费功能

1. 登录 NAS-OS Web 管理界面
2. 进入 **系统设置** → **计费配置**
3. 开启 **启用计费** 开关
4. 配置基础设置：
   - 默认货币（CNY/USD）
   - 计费周期（月结/年结）
   - 账单日（每月几号出账单）

### 2. 配置定价

#### 存储定价

| 配置项 | 说明 | 建议值 |
|--------|------|--------|
| 基础存储价格 | 每GB每周期价格 | ¥0.10/GB/月 |
| SSD 价格 | SSD 存储价格 | ¥0.15/GB/月 |
| HDD 价格 | HDD 存储价格 | ¥0.05/GB/月 |
| 归档价格 | 归档存储价格 | ¥0.02/GB/月 |
| 免费额度 | 免费存储空间 | 100 GB |

#### 带宽定价

| 配置项 | 说明 | 建议值 |
|--------|------|--------|
| 每GB价格 | 流量费用 | ¥0.50/GB |
| 免费额度 | 每月免费流量 | 1000 GB |

### 3. 配置发票信息

设置发票显示的公司信息：

- 公司名称
- 公司地址
- 税号
- 联系电话
- 邮箱

|------|------|----------|

| 周结 | 每周生成账单 | 中型团队 |
| 月结 | 每月生成账单 | 企业标准 |

## 阶梯定价配置

阶梯定价允许根据使用量设置不同价格，鼓励规模化使用。

### 配置示例

```json
[
  {"min_gb": 0, "max_gb": 100, "price_per_gb": 0.15},
  {"min_gb": 100, "max_gb": 500, "price_per_gb": 0.12},
  {"min_gb": 500, "max_gb": 1000, "price_per_gb": 0.10},
  {"min_gb": 1000, "max_gb": null, "price_per_gb": 0.08}
]
```

### 说明

- 0-100 GB：¥0.15/GB
- 100-500 GB：¥0.12/GB
- 500-1000 GB：¥0.10/GB
- 1000+ GB：¥0.08/GB

## 配额管理

### 设置全局配额

1. 进入 **计费配置** → **配额管理**
2. 设置全局上限：
   - 最大存储空间
   - 最大带宽

### 设置用户配额

1. 选择用户
2. 设置该用户的：
   - 存储配额（GB）
   - 带宽配额（GB）
3. 保存配置

### 配额告警

当用户接近配额上限时，系统会：
- 发送通知提醒
- 在仪表板显示警告
- 可配置是否自动限制

---

## 账单管理

### 查看账单

1. 进入 **计费** → **账单管理**
2. 可按状态筛选：
   - 待支付
   - 已支付
   - 已逾期
   - 已作废

### 账单详情

点击账单查看详情：

- 计费周期
- 费用明细
- 税费计算
- 应付金额

### 下载发票

1. 打开账单详情
2. 点击 **下载发票**
3. 获取 PDF 格式发票

---

## 用量统计

### 查看用量

1. 进入 **计费** → **用量统计**
2. 查看当前周期用量：
   - 存储使用量
   - 带宽使用量
   - 服务使用量

### 用量趋势

切换到 **趋势** 标签：
- 日/周/月用量变化
- 预测本月用量
- 成本预测

---

## 成本估算

### 使用成本计算器

1. 进入 **计费** → **成本估算**
2. 输入预期用量：
   - 存储类型（SSD/HDD/归档）
   - 存储量（GB）
   - 带宽（GB）
   - 使用周期
3. 查看估算结果：
   - 月度成本
   - 年度成本
   - 节省建议

---

## 税费配置

### 含税与不含税

- **价格含税**：显示的价格已包含税费
- **价格不含税**：显示的价格不含税，税费单独计算

### 税率设置

1. 进入 **计费配置** → **税费设置**
2. 设置税率（如 13%）
3. 选择税费显示方式

- 提供适度的免费额度吸引用户

- 提供用量实时查询

### 3. 定期审计

- 每月核对账单与实际用量
- 检查异常用量
- 及时处理争议账单

### 4. 预算控制

- 设置配额上限
- 开启用量告警
- 定期审查成本趋势

## 故障排除

### Q: 账单金额不正确？

1. 检查计费配置是否正确
2. 核对用量统计数据
3. 检查阶梯定价配置

### Q: 用户配额不生效？

1. 检查全局配额上限
2. 确认用户配额已保存
3. 检查配额告警是否启用

### Q: 发票信息显示不全？

1. 检查公司信息配置
2. 确认所有必填字段已填写
3. 清除浏览器缓存重试

## API 调用示例

### 获取当前用量

```bash
curl -X GET "https://nas.local/api/v1/billing/usage" \
  -H "Authorization: Bearer <token>"
```

### 获取账单列表

```bash
curl -X GET "https://nas.local/api/v1/billing/invoices" \
  -H "Authorization: Bearer <token>"
```

### 估算成本

```bash
curl -X POST "https://nas.local/api/v1/billing/estimate" \
  -H "Authorization: Bearer <token>" \
  -H "Content-Type: application/json" \
  -d '{"storage_gb": 500, "storage_type": "ssd", "duration_months": 12}'
```

更多 API 详情请参考 [计费 API 文档](../api/billing-api.md)。
