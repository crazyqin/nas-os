# 竞品分析与开发规划

## 竞品调研结果

### 飞牛fnOS 亮点
1. **AI相册增强** - 人物识别、地点/场景智能分类
2. **FN Connect** - 免费内网穿透服务
3. **现代化UI** - 类macOS Dock栏设计
4. **文件管理** - 多设备自动同步
5. **持续迭代** - 无损更新机制

### 群晖DSM 7.3 亮点
1. **冷热数据分层(Tiering)** - 自动迁移冷数据
2. **私有云AI** - 连接各种AI模型，本地LLM支持
3. **AI Console** - 本地PII脱敏
4. **文件共享标签** - 团队协作增强
5. **文件请求** - 安全收集文件
6. **邮件安全审核** - 敏感邮件审批流程

### TrueNAS Scale 亮点
1. **Kubernetes应用** - 企业级容器编排
2. **ZFS高级特性** - ARC缓存优化
3. **GPU转码** - Intel ARC支持
4. **HA高可用** - 多节点故障转移

### TrueNAS Community Edition 25.10 新特性 (2026-03)
1. **RAIDZ Expansion** - 逐盘扩展支持，无需重建阵列即可增加存储容量
2. **Docker替代Kubernetes** - 简化容器管理，更适合家庭用户
3. **全局搜索** - 快速定位文件和应用
4. **GPU Sharing** - GPU资源虚拟化共享，提升AI/转码效率
5. **LXC沙箱支持** - 轻量级容器虚拟化，性能更优
6. **TrueNAS Connect** - 多系统统一管理，一个界面管理多个NAS
