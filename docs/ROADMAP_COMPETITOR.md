# nas-os 功能路线图 (竞品借鉴)

> 基于群晖 DSM、飞牛 fnOS 竞品分析，制定的功能开发计划

## 竞品核心优势分析

### 群晖 Synology DSM
- ✅ Synology Photos: AI 人脸/物体识别、条件相册
- ✅ Active Backup: 企业级整机备份
- ✅ Snapshot Replication: 快照秒级恢复
- ✅ Package Center: 丰富的应用生态
- ✅ 手机 APP: 全平台覆盖

### 飞牛 fnOS
- ✅ 飞牛影视: 自动刮削 99% 识别率
- ✅ Docker 可视化管理
- ✅ FN Connect: 内置内网穿透
- ✅ 云盘挂载: 百度/夸克/阿里云盘
- ✅ 类群晖界面: 新手友好

---

## P0 - 核心功能 (Q2 2026)

### 1. 影视库自动刮削 🔴
**来源**: 飞牛影视

**负责部门**: 兵部(实现) + 礼部(UI)

**功能要点**:
- [ ] 自动识别电影/电视剧
- [ ] 刮削海报、简介、演员信息
- [ ] 支持 TMDB/豆瓣数据源
- [ ] 多端播放支持 (Web/手机/TV)

**技术方案**:
- 后端: Go + TMDB API
- 媒体服务: Jellyfin/Emby 集成
- 前端: React 组件

---

### 2. AI 智能相册 🔴
**来源**: 群晖 Synology Photos

**负责部门**: 兵部

**功能要点**:
- [ ] 人脸识别自动分类
- [ ] 物体/场景识别
- [ ] 地理位置分组
- [ ] 条件相册自动生成
- [ ] RAW 文件支持

**技术方案**:
- AI 引擎: TensorFlow/PyTorch
- 人脸识别: face-recognition
- 场景识别: CLIP 模型

---

### 3. 手机 APP 🔴
**来源**: 群晖 + 飞牛

**负责部门**: 兵部(开发) + 工部(部署)

**功能要点**:
- [ ] iOS/Android 原生应用
- [ ] 照片自动备份
- [ ] 文件管理
- [ ] 影视播放
- [ ] 系统管理

**技术方案**:
- Flutter 跨平台开发
- 或 React Native

---

## P1 - 重要功能 (Q3 2026)

### 4. 应用市场 🟡
**来源**: 群晖 Package Center + 飞牛应用市场

**负责部门**: 工部

**功能要点**:
- [ ] 一键安装应用
- [ ] 官方认证应用库

- [ ] 依赖自动处理

---

### 5. 内置内网穿透 🟡
**来源**: 飞牛 FN Connect

**负责部门**: 工部

**功能要点**:
- [ ] 无需配置即可外网访问

### 6. 快照管理 UI 🟡
