# NatPierce 内网穿透服务隐私合规文档

**文档版本**: v1.0.0  
**生效日期**: 2026-03-25  
**适用产品**: NAS-OS NatPierce 内网穿透服务

## 1. 概述

本文档阐述 NAS-OS NatPierce 内网穿透服务（以下简称"本服务"）的数据处理方式、隐私保护措施及用户权利。本服务旨在为用户提供安全、便捷的远程访问能力，无需公网 IP 即可访问本地 NAS 设备。

### 1.1 服务定义

NatPierce 是 NAS-OS 的内网穿透组件，提供以下功能：
- 远程访问本地 NAS 服务
- 跨网络设备发现与连接
- 安全隧道建立与维护

## 2. 数据传输说明

### 2.1 数据传输类型

| 数据类型 | 用途 | 存储位置 | 加密状态 |
|---------|------|---------|---------|
| 连接凭证 | 身份验证与授权 | 中转服务器内存（临时） | TLS 1.3 加密 |
| 设备标识符 | 设备识别与配对 | 中转服务器内存 | AES-256 加密 |
| 网络流量 | 用户数据传输 | **不存储** | 端到端加密 |
| 连接日志 | 故障排查与服务质量 | 本地设备 | 哈希脱敏 |

### 2.2 数据流向

```
[本地 NAS] <--TLS隧道--> [中转服务器] <--TLS隧道--> [远程客户端]
                    ↑
              仅转发流量
           不存储用户数据
```

### 2.3 第三方数据共享

**我们不会将用户数据出售或分享给第三方**，但以下情形除外：

1. **法律要求**：收到具有法律效力的政府请求时，在法律允许范围内配合
2. **用户授权**：用户明确同意的情况下
3. **服务提供商**：仅限必要的云服务基础设施提供商（如云服务器托管），且均签署保密协议

## 3. 隐私保护措施

### 3.1 技术措施

#### 3.1.1 传输加密
- **协议**: TLS 1.3
- **加密算法**: AES-256-GCM / ChaCha20-Poly1305
- **密钥交换**: ECDHE（前向安全）

#### 3.1.2 认证机制
- **设备认证**: 基于 Ed25519 的设备证书
- **用户认证**: JWT + 多因素认证（MFA）
- **会话管理**: 短期会话令牌，自动刷新

#### 3.1.3 数据最小化
- 中转服务器**不记录用户流量内容**
- 仅记录必要的连接元数据（时间戳、源IP哈希）
- 连接断开后立即清除会话数据

#### 3.1.4 端到端加密选项
- 支持用户自签名证书
- 支持应用层加密（如 HTTPS over Tunnel）
- 中转服务器无法解密用户数据

### 3.2 管理措施

#### 3.2.1 访问控制
- 开发运维人员签署保密协议
- 最小权限原则
- 所有管理操作审计记录

#### 3.2.2 安全审计
- 定期第三方安全审计
- 渗透测试（每季度）
- 漏洞赏金计划

#### 3.2.3 事件响应
- 7×24 安全监控
- 72 小时内安全事件通知
- 应急响应预案

## 4. 用户权利说明

### 4.1 知情权
用户有权了解：
- 收集了哪些个人信息
- 信息的用途
- 信息的存储期限
- 第三方共享情况

### 4.2 访问权
用户可以请求获取：
- 个人设备连接记录
- 账户信息
- 数据处理活动记录

**请求方式**: 
- Web 管理界面 → 设置 → 隐私中心 → 数据导出
- 电子邮件：privacy@nas-os.example.com

### 4.3 更正权
用户可以：
- 更新账户信息
- 修正错误数据
- 删除不准确信息

### 4.4 删除权（被遗忘权）
用户可以请求删除：
- 账户及相关数据
- 连接历史记录
- 设备绑定信息

**注意**: 删除账户将导致服务无法继续使用。

### 4.5 数据可携带权
用户可以导出：
- 设备配置
- 连接历史（CSV/JSON 格式）
- 审计日志

### 4.6 撤回同意权
用户可以随时：
- 撤回数据处理同意
- 停止使用服务
- 注销账户

## 5. Cookie 和追踪技术

### 5.1 使用的技术

| 技术 | 用途 | 存储期限 | 可禁用 |
|-----|------|---------|-------|
| 会话 Cookie | 身份验证 | 浏览器关闭 | 否（必需） |
| 偏好 Cookie | 界面设置 | 1 年 | 是 |
| 本地存储 | 界面状态 | 手动清除 | 是 |

### 5.2 不使用的技术
- 第三方追踪器
- 广告 Cookie
- 社交媒体插件
