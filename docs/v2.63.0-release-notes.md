## 升级说明

### Docker 升级

```bash
# 拉取新版本
docker pull ghcr.io/crazyqin/nas-os:v2.63.0

# 停止旧容器
docker stop nasd

# 启动新容器
docker run -d \
  --name nasd \
  --restart unless-stopped \
  -p 8080:8080 \
  -v /data:/data \
  -v /etc/nas-os:/config \
  ghcr.io/crazyqin/nas-os:v2.63.0
```

### 二进制升级

```bash
# 下载新版本
wget https://github.com/crazyqin/nas-os/releases/download/v2.63.0/nasd-linux-amd64

# 替换旧版本
chmod +x nasd-linux-amd64
sudo mv nasd-linux-amd64 /usr/local/bin/nasd

# 重启服务
sudo systemctl restart nasd
```
