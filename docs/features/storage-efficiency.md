# 存储效率功能对比

## 群晖DSM存储效率功能

### 数据压缩
- 平均节省30%存储空间
- Btrfs原生压缩支持
- 透明压缩，无需手动操作
- 支持多种压缩算法（zstd, lzo）

### 快照管理
- 快速增量快照
- 点恢复功能
- 快照策略调度
- 空间高效（只记录差异）

### SSD缓存
- Btrfs元数据缓存
- 加速文件访问响应
- 写回缓存保护数据
- 混合缓存模式

### 数据去重
- Flash volume deduplication
- 卷级去重
- 5:1数据减少率（虚拟化场景）
- 仅限全SSD卷

### 数据清洗与自愈
- metadata mirroring
- 自动检测文件损坏
- 数据清洗定时任务
- 文件自愈修复

### 自动维护
- Auto Replacement: 热备盘自动克隆故障盘
- Fast Repair: 只修复使用扇区，加速重建

### 加密选项
| 功能 | Volume加密 | 共享文件夹加密 |
|------|-----------|---------------|
| 目标 | 整卷（共享/LUN/包数据） | 共享文件夹 |
| 文件名长度 | 255字符 | 143字符 |
| 密钥存储 | 本地/KMIP | 仅本地 |
| 自动挂载 | 支持 | 支持 |

### Immutable Storage（WriteOnce）
- 防修改/防删除
- 合规归档
- 不可变快照本地/远程复制

## nas-os差异化优势

### Fusion Pool智能分层
群晖有Synology Tiering，但nas-os Fusion Pool更灵活：
- 热数据自动识别
- SSD缓存+HDD容量层
- 用户可配置分层策略

### WriteOnce不可变存储
群晖有类似功能，但nas-os更完善：
- WORM合规归档
- 防勒索软件保护
- 多级别锁定策略
