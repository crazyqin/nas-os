# 竞品分析报告 - 2026-03-25

## 飞牛 fnOS 1.0 (2025年11月发布)

### 核心优势
- **网盘原生挂载**: 115、夸克、123、OneDrive 等直链播放
- **本地AI能力**: 人脸/智能/视频增强识别，人物/宝宝/地图/场景智能相册
- **远程访问**: FN Connect 一键P2P穿透
- **影视体验**: 离线观影、杜比视界、特效字幕、网盘刮削
- **ARM生态**: 此芯P1 Arvm9、小米盒子等，"万物皆可刷飞牛"
- **团队协作**: 团队文件夹、多账号协同、外链分享

### 功能亮点
- 按需唤醒硬盘，延长硬盘寿命
- NTFS/FAT32 外置磁盘直接挂载
- 文件快照、ACL权限体系、iSCSI
- 多RAID、多级SSD缓存、SMART硬盘健康监控
- ext4/ZFS存储池体系

### 弱点
- RAID扩容能力差，无混合硬盘优化（无类似SHR/TRAID）
- 系统稳定性待提升（公测阶段bug频繁）
- 不同容量/类型硬盘混组能力弱

## 群晖 DSM 7.3 (LTS版本，支持到2028年)

### 核心优势
- **Synology Tiering**: 冷热数据分层，大文件读取速度提升30%+
- **安全合规**: 邮件安全审核、标识化保护（个人信息自动脱敏）
- **生态成熟**: 第三方HDD支持、exFAT原生支持
- **文件锁定**: 防止协作冲突
- **稳定可靠**: LTS版本支持周期长

### 技术特点
- DSM 7.3 是许多2016-2019机型的最后版本
- Docker 支持让老机型仍有价值
- 企业级功能完善

## TrueNAS Scale 24.10 Electric Eel / 25.04 Fangtooth

### 核心优势
- **OpenZFS 2.3 集成**: RAIDZ Expansion、Fast Dedup、Direct IO
- **RAIDZ Expansion**: 单盘增量扩容，小系统友好
- **Fast Dedup**: 快速去重，虚拟化场景优化
- **企业级**: 高可用性(HA)、iSCSI、SMB/NFS

### 技术特点
- 基于 Debian Linux
- Docker Compose 原生支持
- ZFS 原生，数据完整性最高

## nas-os 差距分析与优先级

### P0 优先级 (立即行动)
1. **网盘挂载增强** - 飞牛核心卖点，用户需求高
2. **人脸识别/智能相册** - 差异化功能，AI趋势

### P1 优先级 (近期规划)
3. **影视刮削优化** - 网盘刮削、直链播放
4. **移动端体验** - 安卓TV/Apple TV支持

### P2 优先级 (长期规划)
5. **混合硬盘RAID** - 类似SHR的灵活RAID
6. **企业级安全** - 邮件审核、合规功能
