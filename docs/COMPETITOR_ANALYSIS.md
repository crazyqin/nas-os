## 一、飞牛 fnOS 1.0 (2025年11月正式版)

### 产品定位
- 国产免费NAS系统，基于Debian Linux深度开发
- 专注个人及家庭用户，易用性优先
- **1.0版本标志着核心能力基本完善**

### 核心优势（1.0新增）
1. **网盘原生挂载** - 115/夸克/123/OneDrive 原生挂载
2. **本地AI人脸识别** - Intel核显加速，人物/宝宝/地图/场景智能相册
3. **FN Connect** - 三网稳定远程访问，免费内网穿透
4. **虚拟机硬件直通** - 一机多用
5. **ext4/ZFS存储池** - 多级SSD缓存、SMART监控
6. **半月更节奏** - 快速迭代

### 文件系统与存储
- ext4/ZFS 存储池体系
- 多级 SSD 缓存
- SMART 硬盘健康监控
- 文件快照、ACL 权限体系、iSCSI
- NTFS/FAT32 外置磁盘直接挂载
- **按需唤醒硬盘** - 延长硬盘寿命

### 多媒体影音
- 网盘刮削和直链播放
- 杜比视界、特效字幕
- 离线观影、跳过片头片尾

### 相册备份
- 苹果/安卓多品牌照片/动图无损备份
- 人脸识别 V1.0.1（Intel核显加速）
- 相似/重复查重机制
- 视频转码预览

### 学习借鉴（高优先级）
- 🔴 **网盘原生挂载** - 115/夸克/阿里云盘 (P0)
- 🔴 **本地AI人脸识别** - Intel核显加速 (P0)
- 🟡 **按需唤醒硬盘** - 节能特性 (P1)
- 🟡 **网盘刮削直链播放** (P1)

## 二、群晖 Synology DSM 7.3 (2025年10月正式版)

### 产品定位
- 企业级NAS解决方案，功能最全面
- 高端市场领导者

### 核心优势（7.3新增）
1. **Synology Tiering 分层存储** - 智能"热"数据高性能、"冷"数据低成本
   - 依据存取频率与行为自动分层
   - 自定义分层策略（修改时间/存取频率）
   - 去重优化、勒索软件防护
2. **AI Console 数据脱敏** - 发送敏感数据前本地遮罩
3. **Synology Drive 4.0** - 共享标签、文件请求、文件锁定
4. **MailPlus 邮件审核** - 域共用功能、跨站点协同
5. **exFAT 原生支持** - 无需额外购买授权
6. **安全升级** - KEV/EPSS/LEV风险指标，50+安全更新/年

### 学习借鉴（高优先级）
- 🔴 **Synology Tiering** - 我们已实现Fusion Pool (✅)
- 🔴 **文件锁定机制** - 避免协作冲突 (P0)
- 🟡 **AI Console数据脱敏** (P1)
- 🟡 **共享标签系统** (P1)

## 三、TrueNAS Scale Dragonfish

### 产品定位
- 开源免费，企业级ZFS存储
- 技术爱好者和小型企业首选

### 核心优势
1. **ZFS文件系统** - 数据完整性保障
2. **原生Netdata监控** - 系统级性能监控
3. **应用生态** - Kubernetes原生应用支持
4. **SMB/NFS审计日志** - 企业合规

### 核心功能
- 存储池管理
- 快照与复制
- 应用目录（官方/社区）
- 原生监控集成
