# NAS-OS v2.18.0 发布说明

**发布日期**: 2026-03-15
**版本号**: v2.18.0
**基于版本**: v2.17.1

---

## 📝 版本概述

v2.18.0 是一个功能完善版本，重点提升代码质量和测试覆盖率，同时新增下载器集成和照片管理增强功能。

---

## 📋 变更内容

### Added (新增)

- **下载器模块**
  - Transmission BT 下载器集成
  - qBittorrent 下载器集成
  - 统一下载任务管理 API
  - WebUI 下载管理页面 (`webui/pages/downloader/`)

- **照片管理增强**
  - 分片上传支持大文件
  - 智能搜索功能
  - AI 照片分析增强

- **数据分层策略完善**
  - 自动数据迁移规则
  - 存储层监控增强

- **网络配置持久化**
  - 网络配置保存和恢复
  - 配置版本管理

### Changed (变更)

- **文档更新**
  - README.md 版本号更新至 v2.18.0
  - CHANGELOG.md 添加 v2.18.0 条目
  - API_GUIDE.md 版本号更新
  - api.yaml 版本号更新

- **测试覆盖率提升**
  - 单元测试完善
  - 集成测试增强
  - 覆盖率达到 40%+

- **国际化完善**
  - 中英文语言包更新

## 📚 功能亮点

### 下载器集成

支持主流 BT 下载器：
- **Transmission**: 轻量级、低资源占用
- **qBittorrent**: 功能丰富、WebUI 完善

### 照片管理

智能照片管理功能：
- 分片上传支持超大照片
- AI 自动分类和标签
- 智能搜索和过滤

### 测试质量

测试覆盖率显著提升：
- 单元测试覆盖核心模块
- 集成测试验证端到端流程
- 目标覆盖率 40%+
