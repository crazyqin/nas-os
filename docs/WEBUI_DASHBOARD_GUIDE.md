# WebUI 仪表板使用说明

**版本**: v2.2.0  
**更新日期**: 2026-03-21

---

## 📖 概述

NAS-OS WebUI 仪表板是系统的主要控制中心，提供实时系统监控、快速操作入口和关键信息展示。通过直观的可视化界面，用户可以快速了解系统状态并执行常用操作。

### 特性

- ✅ 实时系统资源监控
- ✅ 可自定义的小部件布局
- ✅ 快速操作面板
- ✅ 活动告警展示
- ✅ 最近任务历史
- ✅ 响应式设计（支持移动端）

---

## 🖥️ 访问仪表板

### 登录

1. 打开浏览器访问 NAS-OS 地址：`http://<NAS_IP>:8080`
2. 输入用户名和密码
3. 登录后自动进入仪表板

### 首次登录

默认凭据：
- 用户名：`admin`
- 密码：`admin123`

⚠️ **重要**: 首次登录后请立即修改默认密码！

## 📦 小部件说明

### 系统概览小部件

显示实时系统资源使用情况：

| 指标 | 说明 | 刷新频率 |
|------|------|----------|
| CPU 使用率 | 所有核心的平均使用率 | 1 秒 |
| 内存使用率 | 已用内存占总内存的百分比 | 1 秒 |
| 磁盘使用率 | 所有磁盘的平均使用率 | 10 秒 |
| 系统运行时间 | 系统启动至今的时间 | 60 秒 |

**颜色指示**:
- 🟢 绿色：正常（< 70%）
- 🟡 黄色：警告（70% - 90%）
- 🔴 红色：危险（> 90%）

### 存储健康小部件

显示存储设备状态：

| 信息 | 说明 |
|------|------|
| 状态指示 | 整体存储健康状态 |
| 在线磁盘 | 在线磁盘数量/总磁盘数 |
| 温度 | 最高磁盘温度 |
| 错误计数 | SMART 错误总数 |

### 网络流量小部件

显示实时网络流量：

| 指标 | 说明 |
|------|------|
| 上传速度 | 当前上传速度 |
| 下载速度 | 当前下载速度 |
| 接口名称 | 主要网络接口 |

### 资源使用趋势小部件

显示历史资源使用趋势图：

- 时间范围：可切换 1 小时 / 24 小时 / 7 天
- 图表类型：折线图
- 支持缩放和拖动

### 快速操作面板

常用操作快捷入口：

| 按钮 | 功能 |
|------|------|
| + 卷 | 创建新存储卷 |
| 快照 | 创建快照 |
| 用户 | 创建新用户 |
| 共享 | 创建文件共享 |

### 活动告警小部件

显示未处理的系统告警：

- 按严重程度排序（Critical > Warning > Info）
- 点击告警可查看详情
- 支持确认和忽略操作

### 最近任务小部件

显示最近执行的任务：

| 信息 | 说明 |
|------|------|
| 任务类型 | 快照/备份/同步等 |
| 状态 | 运行中/成功/失败 |
| 时间 | 执行时间 |
| 详情 | 点击查看详情 |

## ⚙️ 自定义仪表板

### 添加小部件

1. 点击仪表板右上角 **添加小部件** 按钮
2. 从小部件库中选择要添加的小部件
3. 拖动小部件到目标位置
4. 点击 **保存布局**

### 移除小部件

1. 将鼠标悬停在小部件上
2. 点击右上角的 **×** 按钮
3. 确认移除

### 调整布局

- 拖动小部件标题栏可移动位置
- 拖动小部件右下角可调整大小
- 布局会自动保存

### 重置布局

1. 点击仪表板右上角 **设置** 图标
2. 选择 **重置为默认布局**
3. 确认重置

---

## 📱 移动端适配

### 响应式布局

仪表板会根据屏幕尺寸自动调整布局：

- **桌面端**: 多列网格布局
- **平板端**: 双列布局
- **手机端**: 单列垂直布局

### 触摸操作

- 下拉刷新数据
- 左右滑动切换页面
- 长按小部件进入编辑模式

### PWA 支持

NAS-OS 支持 PWA (Progressive Web App)：

1. 在移动浏览器中访问 NAS-OS
2. 点击浏览器菜单中的 **添加到主屏幕**
3. 从主屏幕启动获得原生应用体验

## 🔔 通知中心

### 访问通知中心

点击右上角 🔔 图标打开通知中心。

### 通知类型

| 类型 | 说明 |
|------|------|
| 系统 | 系统事件通知 |
| 告警 | 系统告警通知 |
| 任务 | 任务完成通知 |
| 安全 | 安全相关通知 |

### 通知设置

1. 打开 **设置** → **通知**
2. 配置通知类型和渠道
3. 设置通知静默时段

---

## 🎨 主题设置

### 切换主题

1. 点击右上角用户头像
2. 选择 **主题设置**
3. 选择主题：
   - 浅色主题
   - 深色主题
   - 跟随系统

### 自定义配色

高级用户可自定义界面配色：

1. 打开 **设置** → **外观**
2. 选择自定义配色方案
3. 调整颜色值
4. 点击 **应用**

---

## 📊 数据导出

### 导出系统报告

1. 点击仪表板右上角 **导出** 按钮
2. 选择报告类型：
   - PDF 报告
   - CSV 数据
   - JSON 数据
3. 选择时间范围
4. 点击 **下载**

### 计划报告

可设置定期自动生成报告：

1. 打开 **设置** → **报告**
2. 配置报告计划
3. 选择发送方式（邮件/下载）

## 🔧 故障排查

### 数据不更新

1. 检查 WebSocket 连接状态
2. 刷新页面
3. 检查浏览器控制台错误

### 布局无法保存

1. 检查浏览器存储权限
2. 清除浏览器缓存
3. 尝试使用隐私模式

### 性能问题

1. 减少小部件数量
2. 降低刷新频率
3. 检查网络连接
