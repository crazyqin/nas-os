## 模块1: AI去敏感化器

### 功能特性
- **敏感信息自动识别**: 支持多种敏感数据类型
  - 信用卡号 (Luhn校验)
  - 身份证号 (校验位验证)
  - 手机号码
  - 电子邮箱
  - IP地址
  - 密码/API密钥
  - 银行账号
  - 护照号码

- **多种去敏感模式**:
  - `redact`: 完全移除
  - `mask`: 部分遮蔽
  - `replace`: 替换为占位符
  - `encrypt`: AES-256-GCM加密
  - `hash`: SHA256哈希
  - `tokenize`: 令牌化

- **本地化处理**: 所有数据在本地处理，不上传云端

### 安全特性
- AES-256-GCM加密支持
- 令牌存储支持过期时间
- Luhn算法验证信用卡
- 身份证校验位验证
- 自定义敏感词和模式支持

## 模块2: 不可变快照管理

### 功能特性
参考 TrueNAS Electric Eel 和群晖 DSM 7.3 实现:

- **多级锁定机制**:
  - `soft`: 软锁（可解锁）
  - `hard`: 硬锁（需条件解锁）
  - `timed`: 时间锁（定时解锁）
  - `permanent`: 永久锁（不可解锁）
  - `compliance`: 合规锁

- **勒索软件防护**:
  - WORM (Write Once Read Many) 支持
  - 异常访问检测
  - 完整性验证
  - 自动锁定保护

- **时间锁定机制**:
  - 可配置锁定期限
  - 自动过期解锁
  - 延长锁定支持

- **保留策略**:
  - 最小/最大保留期
  - 按时间周期保留（日/周/月/年）
  - 数量限制

### 安全特性
- 完整性校验 (SHA256)
- 访问日志记录
- 审计追踪
- 早释放审批机制
- 最小保留期强制执行
